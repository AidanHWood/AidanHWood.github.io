Password Manager

A basic password manager application built in Python, designed to securely store and manage user credentials.

The project utilizes encryption via the cryptography.fernet library to protect sensitive data, ensuring passwords are stored safely.

The user interface is created with CustomTkinter, providing a clean and intuitive GUI for adding, viewing, and managing passwords.

This project helped strengthen understanding of Python programming, data security practices, and GUI development.
